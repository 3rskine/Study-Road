## 探索領域

### 1. 視覺化與即時渲染 (3D / 遊戲 / AR/VR)

**核心技能**: 3D 幾何、材質、光照、GPU/著色器

**工具**: C++、GLSL、OpenGL / Vulkan、Unity (C#)、Unreal (C++)

**專案建議**:

#### 1.1 旋轉立方體渲染器
- **描述**: 用 OpenGL 從零寫一個 3D 立方體，能用滑鼠旋轉、加入 Phong 光照模型(環境光+漫射+鏡面反射)
- **時間**: 2-3 天
- **交付物**: 可執行檔 + 原始碼 + 控制說明

#### 1.2 Unity 迷你遊戲場景
- **描述**: 在 Unity 建立一個第三人稱角色，能用 WASD 移動、滑鼠控制相機、加入簡單地形和障礙物
- **時間**: 3-4 天
- **交付物**: 可玩的 Build 檔 + 1 分鐘遊玩影片

#### 1.3 客製化 Shader 效果
- **描述**: 寫一個 Fragment Shader 做卡通渲染(cel-shading)或水波紋效果，並套用到 3D 模型上
- **時間**: 2-3 天
- **交付物**: Shader 程式碼 + 前後對比影片

**驗證標準**: 如果你調整光影參數時停不下來，且想研究更複雜的渲染技術 → 值得深入

### 2. 軟體工程 & 網頁 (後端/前端/全端)

**核心技能**: 系統設計、API、部署、版本控制、測試

**工具**: JavaScript/TypeScript、Node.js、React、Git、Docker

**專案建議**:

#### 2.1 待辦事項全端 App
- **描述**: 用 React 做前端、Express + PostgreSQL 做後端，實作新增/刪除/編輯/完成待辦事項，支援多使用者登入
- **時間**: 5-7 天
- **交付物**: GitHub Repo + 本地可運行說明 + API 文件

#### 2.2 Docker 容器化部署
- **描述**: 將上述 App 寫成 Dockerfile 和 docker-compose.yml，一鍵啟動前後端和資料庫，並部署到 Render 或 Railway
- **時間**: 2-3 天
- **交付物**: 部署連結 + docker-compose 檔案 + 部署文件

#### 2.3 CI/CD 與測試流程
- **描述**: 用 GitHub Actions 設定自動測試(Jest 前端 + Supertest 後端)，並在 push 時自動部署
- **時間**: 2-3 天
- **交付物**: .github/workflows 設定檔 + 測試覆蓋率報告

**驗證標準**: 如果你享受把想法快速做成可用產品，且願意持續優化架構 → 值得深入

### 3. 韌體與嵌入式系統

**核心技能**: 硬體介面、即時性、低層驅動、資源受限程式設計

**工具**: C / C++、FreeRTOS、ESP32 / STM32、I²C/SPI/UART

**專案建議**:

#### 3.1 溫濕度監測裝置
- **描述**: 用 ESP32 + DHT22 感測器讀取溫濕度，每 10 秒在序列埠顯示數值，並用 OLED 螢幕即時顯示
- **時間**: 2-3 天
- **交付物**: 電路圖(Fritzing) + 程式碼 + demo 影片

#### 3.2 馬達 PID 控制
- **描述**: 用 Arduino + L298N 馬達驅動板控制直流馬達，實作 PID 控制讓馬達穩定在目標轉速(用編碼器回饋)
- **時間**: 3-4 天
- **交付物**: 程式碼 + PID 參數調整紀錄 + 轉速曲線圖

#### 3.3 IoT 雲端上傳
- **描述**: 將感測器資料透過 MQTT 或 HTTP POST 上傳到 ThingSpeak/Firebase，並做簡單的網頁儀表板視覺化
- **時間**: 3-4 天
- **交付物**: 完整系統架構圖 + 雲端儀表板連結 + 程式碼

**驗證標準**: 如果硬體 debug 讓你有成就感，且享受讓實體裝置動起來 → 值得持續

### 4. 資料工程與資料庫

**核心技能**: 資料建模、查詢效能、資料管道(ETL)、批流合併處理

**工具**: SQL、PostgreSQL、Airflow、Spark、Python

**專案建議**:

#### 4.1 電商資料庫設計
- **描述**: 設計一個正規化的電商資料庫(使用者、商品、訂單、評論)，寫出 10 個常見查詢(例如：月銷售排行、使用者購買歷史)
- **時間**: 3-4 天
- **交付物**: ER Diagram + SQL schema + 查詢範例 + 說明文件

#### 4.2 CSV 自動匯入報表系統
- **描述**: 用 Python 寫一個 ETL 腳本，從 CSV 清洗資料(處理缺失值、格式化)、匯入 PostgreSQL、自動產生 HTML 銷售報表
- **時間**: 4-5 天
- **交付物**: Python 腳本 + 範例 CSV + 產出的報表 HTML

#### 4.3 Spark 大資料處理
- **描述**: 用 PySpark 處理一個大型 CSV(>100MB)，做 GroupBy 聚合計算、篩選異常值，並輸出成 Parquet 格式
- **時間**: 3-4 天
- **交付物**: Spark 程式碼 + 執行時間比較(Pandas vs Spark) + 結果檔案

**驗證標準**: 如果你喜歡整理混亂資料並找出商業洞察 → 值得深入

### 5. 網路與通訊

**核心技能**: 協定棧(TCP/IP)、封包分析、安全性、延遲/頻寬考量

**工具**: Linux、Wireshark、socket 程式、iptables、BPF

**專案建議**:

#### 5.1 Socket 檔案傳輸工具
- **描述**: 用 Python 或 C 寫一個 client-server 程式，能透過 TCP 傳送任意檔案，並顯示傳輸進度百分比
- **時間**: 3-4 天
- **交付物**: 程式碼 + 使用範例 + 傳輸速度測試結果

#### 5.2 Wireshark 封包分析報告
- **描述**: 用 Wireshark 抓取自己瀏覽 HTTPS 網站的封包，分析 TLS 握手過程，畫出時序圖並解釋每個步驟
- **時間**: 2-3 天
- **交付物**: Wireshark 截圖 + 時序圖 + 分析報告(PDF)

#### 5.3 防火牆規則實驗
- **描述**: 在 Linux VM 用 iptables 設定規則：只允許特定 IP 的 SSH 連線、封鎖所有 HTTP、允許 HTTPS，並測試驗證
- **時間**: 2-3 天
- **交付物**: iptables 規則列表 + 測試截圖 + 說明文件

**驗證標準**: 如果你對封包底層運作充滿好奇，且享受網路除錯 → 值得深入
